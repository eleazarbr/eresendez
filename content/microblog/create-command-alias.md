Algo más que no quiero recordar: ¿Cómo crear alias en mi terminal?

```bash
# Create alias in ubuntu
sudo nano ~/.bashrc
alias aliasname='commands'
source ~/.bashrc

# Create alias in mac
sudo nano .bash_profile
alias aliasname='commands'
source .bash_profile

sudo nano .zshrc
alias aliasname='commands'
```
